# Appose Fiji Plugin Example

An example Fiji plugin that demonstrates how to call native Python 
code from Java using [Appose](https://github.com/apposed/appose). 
This plugin rotates an image by 90 degrees using scikit-image in Python 
and returns the result to Fiji.

## Requirements

- Java 21 or higher
- Fiji/ImageJ (tested with recent versions)
